# Platzhalter — Claude Code Instructions

Read `AGENTS.md` for full development rules. Read `DESIGN.md` for architecture and design decisions.

## Quick Reference

- **Dev environment**: `devbox shell` (never install global packages)
- **Start services**: `devbox services up` (postgres, mailpit, backend, frontend)
- **Package manager**: `bun` (not npm). Use `bunx` instead of `npx`.
- **Frontend**: Next.js App Router at `/frontend` (port 3000)
- **Backend**: Go Fiber at `/backend` (port 8080)
- **Database**: PostgreSQL (port 5432, db: `platzhalter`)
- **Mail**: Mailpit (SMTP 1025, UI 8025)

## Key Constraints

- No ORM — use sqlc for database queries, golang-migrate for migrations
- No business logic in Next.js API routes — all API logic in Go backend
- No `npm`/`npx` — use `bun`/`bunx`
- No JWT in localStorage — HttpOnly cookies only
- No hover-only UI — mobile-first, touch targets >= 44x44pt
- Don't edit `backend/internal/db/queries/` — auto-generated by `sqlc generate`

## Critical Paths

| What                | Path                                  |
|---------------------|---------------------------------------|
| Backend entrypoint  | `backend/cmd/server/main.go`          |
| Backend config      | `backend/internal/config/config.go`   |
| DB connection       | `backend/internal/db/db.go`           |
| Route handlers      | `backend/internal/handler/`           |
| Middleware          | `backend/internal/middleware/`         |
| Migrations          | `backend/migrations/`                 |
| sqlc queries        | `backend/sqlc/query.sql`              |
| sqlc config         | `backend/sqlc.yaml`                   |
| Frontend config     | `frontend/next.config.ts`             |
| API client          | `frontend/src/lib/api.ts`             |
| Auth client         | `frontend/src/lib/auth.ts`            |
| i18n messages       | `frontend/src/messages/`              |
| UI components       | `frontend/src/components/ui/`         |
| Environment vars    | `.env` (not committed)                |
| Service orchestration | `process-compose.yaml`              |
